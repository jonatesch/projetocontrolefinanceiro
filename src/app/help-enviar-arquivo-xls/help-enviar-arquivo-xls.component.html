Instruções para envio de arquivo

Envie um arquivo .xls ou .xlsx que contenha o seguinte cabeçalho:



A última coluna, efetuada, é opcional.

Caso o arquivo não contenha essa coluna, todas as movimentações serão inseridas como efetuadas.

As colunas, categoria, origem e orçamento são obrigatórias, mas as linhas podem estar em branco.

Caso o arquivo não contenha informação nessas colunas, todas as movimentações serão inseridas com valores padrão para essas colunas:

  • Categoria: outros
  • Origem: Padrão
  • Orçamento: Normal
A coluna mês de ref deve conter informação no formato YYYY/MM, por exemplo: 202201 para indicar Janeiro de 2022.

Se o mês de ref das suas movimentações sempre coincide com o mês da data, você pode utilizar a seguinte fórmula para gerar automaticamente o mês de ref:

=IF(A2;YEAR(A2)&TEXT(A2;"MM");"")
=SE(A2;ANO(A2)&TEXTO(A2;"MM");"")

Copie de acordo com o idioma do seu sistema operacional.
